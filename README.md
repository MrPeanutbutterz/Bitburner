# Bitburner

All scripts for Bitburner

GENESIS: 

HackServer: Weakens the security, grows the balance, hacks the money avaliable, runs incremental on every hack + 1%
Break: Weakens the security of a server, use arg to define target server with a delay
Spoof: Grows the money of a server, use arg to define target server with a delay
Collect: Hacks the money available of a server, use arg to define target server with a delay
